# Naver Clone

Responsive Naver-inspired portal built with React and TypeScript.

## Features
- Search bar UI
- News section
- Weather API integration
- Trend ranking
- Responsive layout

## Tech Stack
- React
- TypeScript
- Vite
- REST API
